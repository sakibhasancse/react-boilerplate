# Coxtunes lab

## Getting started
### Project install

- To install the project first you need to clone the project, open your terminal and run the following command

```sh
> git clone https://gitlab.com/coxtuneslab/coxtuneswebsite
> cd coxtuneswebsite
```

## Running the application

```
npm install 
npm start
or
yarn
yarn start
```

## Add your files

- [ ] [Create](https://docs.gitlab.com/ee/user/project/repository/web_editor.html#create-a-file) or [upload](https://docs.gitlab.com/ee/user/project/repository/web_editor.html#upload-a-file) files
- [ ] [Add files using the command line](https://docs.gitlab.com/ee/gitlab-basics/add-file.html#add-a-file-using-the-command-line) or push an existing Git repository with the following command:

```
git remote add origin https://gitlab.com/coxtuneslab/coxtuneswebsite
git branch -M main
git push -u origin main
```
